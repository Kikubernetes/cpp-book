# C++ のバージョン

C++ には以下のバージョンがあります。本書では C++11 を対象とします。

* C++98 (1998年制定)
* C++03 (2003年制定)
* [C++11] (2011年制定)
* [C++14] (2014年制定)
* [C++17] (2017年制定)

2020年には [C++20] が制定される予定です。

[C++14]: https://cpprefjp.github.io/lang/cpp14.html
[C++17]: https://cpprefjp.github.io/lang/cpp17.html
[C++20]: https://cpprefjp.github.io/lang/cpp20.html
